## Com Compilar
1. Ara per ara, només caldrà netejar anteriors execucions:
   ```sh
   make clean
   ```
2. Compilar novament emprant la comanda make:
   ```sh
   make
   ```
3. Per executar emprar el executable p1 de la següent manera:
   ```sh
   ./p1 <operacio> <nom_volum>
   ```
